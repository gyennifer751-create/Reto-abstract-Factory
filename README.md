# Reto-abstract-Factory

### El Escenario
Debes implementar un sistema de UI que soporte **Windows** y **MacOS**. El código cliente ya está configurado en `Application.java`, pero actualmente no funciona porque las fábricas y los productos están vacíos.

### Tu Misión:
1.  **Implementar los Productos:** Ve a la carpeta `products/` y haz que cada botón/checkbox imprima su nombre (ej: "Renderizando Botón de Windows").
2.  **Implementar las Fábricas:** En `factories/`, haz que cada fábrica devuelva el objeto correcto de su familia.
3.  **No tocar la lógica de `Application`:** El objetivo es que la app funcione sin saber qué SO está usando.

### Cómo empezar:
Busca los comentarios `// TODO: Implementar aquí` en el código.
